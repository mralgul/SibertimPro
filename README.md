# 🛡️ SiberTimPro - Akıllı Wordlist & Hash Aracı

Adı SiberTimPro olan projem Python dili ile geliştirilmiş, sızma testleri (pentest) ve şifre kırma (cracking) işlemlerinde kullanılabilen etkili bir siber güvenlik aracıdır. Bu projeyi geliştirirken hem sızma testi hem de kriptoloji alanlarında kendimi geliştirme fırsatı buldum. 

## 🚀 Temel Özellikler

Bu araç üç temel fonksiyona ev sahipliği yapmaktadır:

* **🧠 Akıllı Wordlist Üretme:** Sızma testleri için hedef odaklı (isim, sayı, sembol ve özel kelime kombinasyonları) akıllı varyasyonlar ve "Leetspeak" formatında şifre listeleri oluşturma.
* **🔒 Kapsamlı Hashleme:** Girilen metinleri anlık olarak **MD5, SHA1, SHA256, SHA384, SHA512 ve CRC32** algoritmalarıyla şifreleme.
* **🔓 Gelişmiş Hash Çözme:** Tekil varyasyonlar üzerinden veya sisteme yüklenen bir `.txt` wordlist dosyası üzerinden toplu hash çözümleme seçenekleri.
* **💾 Dışa Aktarma:** Oluşturduğunuz wordlist, hash veya hash çözüm sonuçlarını raporlamak üzere **.txt** veya **.zip** formatlarında dışa aktarma imkanı.
