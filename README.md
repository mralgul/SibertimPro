# 🛡️ SiberTimPro - Akıllı Wordlist & Hash Aracı

Adı SiberTimPro olan projem Python dili ile geliştirilmiş, sızma testleri ve şifre kırma işlemlerinde kullanılabilen etkili bir siber güvenlik aracıdır. Bu projeyi geliştirirken hem sızma testi hem de kriptoloji alanlarında kendimi geliştirme fırsatı buldum. Bundan sonraki süreçte de edindiğim bu değerli deneyimleri daha ileri seviyelere taşımayı hedefliyorum.

## 🚀 Temel Özellikler

Bu araç üç temel fonksiyona ev sahipliği yapmaktadır:

* **🧠 Wordlist Üretme:** Sızma testleri için akıllı varyasyon oluşturma.
* **🔒 Hashleme:** Girilen metinleri MD5, SHA1, SHA256, SHA384, SHA512, CRC32 formatlarında şifreleme.
* **🔓 Hash Çözme:** Tekil veya dosyada toplu hash çözümü seçenekleri.
* **💾 Dışa Aktarma:** Oluşturduğunuz Wordlist, Hash veya Hash çözümlerini `.txt` veya `.zip` formatlarında dışa aktararak kullanma imkanı.

Projeyi kendi bilgisayarınızda çalıştırmak için aşağıdaki adımları izleyebilirsiniz:

1. Repoyu bilgisayarınıza indirin (Klonlayın):
   ```bash

2. Proje dizinine gidin:
   ```bash
   cd SibertimPro

3. Gerekli kütüphaneleri yükleyin (Arayüzde görsel kullanımı için Pillow gereklidir):
   ```bash
   pip install Pillow

4. Aracı başlatın:
   ```bash
   python sibertimpro.py
